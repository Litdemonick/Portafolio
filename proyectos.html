Simulación de Ataques en Red Local

Informe técnico que documenta la ejecución de escaneos de puertos y simulaciones de ataques de denegación de servicio (DoS) en un entorno controlado, con el fin de analizar vulnerabilidades y monitorear el tráfico de la red.

Examen Parcial: Infraestructura y Desarrollo Web

Evaluación práctica de la asignatura Desarrollo de Software IV que solicita la implementación de un servidor NGINX en Linux , el diseño de un diagrama de red para interconectar dos oficinas provinciales con equipos específicos y la creación de un sitio web funcional para el Centro Especializado en Lenguas.

Ejercicio Práctico 1: Desarrollo Web y Servidores Linux

Informe técnico presentado por Carlos Miranda y Mariam Harris que documenta la creación de un sitio web seguro con facturación PDF para la pyme "SUCESOS y MÁS", la configuración de un servidor Ubuntu con acceso remoto SSH/FTP y la implementación de un entorno XAMPP funcional.

Investigación 3: Replicación de Base de Datos MySQL/MariaDB

Presentación y guía de asignación desarrollada por los estudiantes Carlos Miranda y Harold Morales para la asignatura Desarrollo de Software IV. El proyecto detalla la implementación técnica de una replicación de base de datos tipo Maestro-Esclavo entre sistemas operativos Windows y Ubuntu utilizando MySQL/MariaDB, PHP y XAMPP. Se documenta la configuración específica de puertos y archivos de sistema (my.ini, my.cnf) para lograr la sincronización de datos en tiempo real, validada mediante una aplicación web demostrativa.

Taller Práctico 3: Sistema Integral para Chinos Café

Proyecto presentado por Carlos Miranda y Brayan Quintero que detalla una solución tecnológica para la empresa "Chinos Café". El trabajo integra el desarrollo de un sistema de Punto de Venta (POS) web gestionado con MySQL, el diseño de red para interconectar dos sucursales mediante VLANs y la implementación de seguridad con Firewall y Fail2ban para la mitigación de ataques.

Examen Parcial 2: Sistema de Mantenimiento Web

Evaluación que solicita el desarrollo de una solución web para la gestión de mantenimiento de la pyme "NIBARRA", incluyendo operaciones CRUD con replicación de base de datos, calendario de agenda, seguimiento de estados tipo Kanban y un Chatbot. Además, requiere el diseño de una red LAN que conecte la operación en Chiriquí con un Data Center en Panamá.

Examen Final: Portal de Vacantes y Arquitectura de Red

Proyecto final para la empresa "Consultores Chiriquí, S.A." que requiere el desarrollo de un sitio web para la gestión de vacantes públicas y privadas (sin necesidad de hoja de vida), integrando un chatbot para medir interacciones y un sistema de facturación basado en peajes digitales. La solución exige una infraestructura técnica compleja con servidores Apache/NGINX, replicación de bases de datos en tiempo real y el diseño de una red LAN que interconecte las sedes de Chiriquí y Panamá.
